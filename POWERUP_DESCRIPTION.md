# Power-Up Description for Trello Review

## Overview (short one-liner):
Import JSON and CSV files into Trello boards. Map columns to cards with one click.

## Description (full):

**Easily import data from JSON or CSV files directly into your Trello board.**

### What it does:
- **Drag & drop** your JSON or CSV file
- **Smart column mapping** - automatically detects title, list, and description columns
- **Creates cards** in existing Trello lists or creates new lists automatically
- **Supports multiple formats** - flat JSON, nested JSON, Trello board exports, and CSV files

### Perfect for:
- Migrating data from other project management tools
- Bulk card creation from spreadsheets
- Importing Trello board backups
- Converting CSV exports into Trello cards

### How it works:
1. Click **"Import JSON/CSV"** button on your board
2. Drop your file or click to browse
3. Map your columns to card fields (title, list, description)
4. Click **Import** - cards are created instantly

### Features:
- ✅ Supports JSON arrays and nested structures
- ✅ CSV parsing with automatic header detection
- ✅ Smart list matching (case-insensitive)
- ✅ Option to create missing lists automatically
- ✅ Progress tracking during import
- ✅ Error handling with detailed feedback
